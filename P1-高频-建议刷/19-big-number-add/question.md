# 19 - 大数相加

## 基本信息

| 项目 | 内容 |
|------|------|
| 分类 | 数据结构与算法 |
| 难度 | ⭐⭐⭐ |
| 考察点 | 字符串运算、进位处理、数学思维 |

## 背景

JavaScript 中 `Number` 类型采用 IEEE 754 双精度浮点数，安全整数范围为 `-(2^53 - 1)` 到 `2^53 - 1`。当整数超过这个范围时会丢失精度：

```javascript
9007199254740992 + 1  // → 9007199254740992（丢失精度！）
```

因此对于超大整数的加法运算，需要通过字符串模拟竖式加法来实现。

## 题目要求

实现一个函数 `add(a, b)`：

- **参数**：两个字符串 `a` 和 `b`，表示非负大整数（可能非常长）
- **返回值**：字符串，表示两个大整数的和
- **约束**：不使用 `BigInt`，必须通过字符串模拟竖式加法实现

## 示例

```javascript
add('999999999999999999', '1')   // → '1000000000000000000'
add('123', '456')                 // → '579'
add('999', '1')                   // → '1000'
add('0', '0')                     // → '0'
```

## 约束

1. 不使用 `BigInt`，必须通过字符串逐位计算
2. 考虑边界情况：空字符串、前导零（如 `'007'`）、两个数长度差异大
3. 时间复杂度应为 O(max(m, n))，m、n 分别为两个输入字符串的长度
