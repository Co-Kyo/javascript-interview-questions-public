# 08 - 数组扁平化 flat（支持指定深度）

## 题目信息

- **分类**：JavaScript 核心手写
- **难度**：⭐⭐
- **考察点**：递归、reduce、迭代实现、数组操作

---

## 背景

在实际开发中，我们经常需要处理嵌套的数组结构：

- **API 返回的嵌套数据**：后端返回的树形结构需要展开为扁平列表
- **DOM 树遍历**：将嵌套的节点列表展平为一维数组
- **路由配置**：嵌套的路由配置需要展平为路由表
- **文件目录**：递归遍历目录结构

JavaScript 原生提供了 `Array.prototype.flat()` 方法，但面试中要求你手写实现，并支持指定深度。

---

## 要求

在 `Array.prototype` 上实现 `myFlat(depth)` 方法：

1. **参数 `depth`**：指定扁平化的嵌套深度，默认值为 `1`
2. **`depth = 1`**：只展开一层嵌套
3. **`depth = Infinity`**：完全展开所有嵌套层级
4. **`depth = 0`**：返回原数组的浅拷贝，不展开任何层级
5. **不使用** `Array.prototype.flat`（原生方法）

---

## 示例

```javascript
// 示例 1：展开一层
[1, [2, [3, [4]]]].myFlat(1)
// → [1, 2, [3, [4]]]

// 示例 2：完全展开
[1, [2, [3, [4]]]].myFlat(Infinity)
// → [1, 2, 3, 4]

// 示例 3：不展开
[1, [2, [3]]].myFlat(0)
// → [1, [2, [3]]]（浅拷贝）

// 示例 4：展开两层
[1, [2, [3, [4]]]].myFlat(2)
// → [1, 2, 3, [4]]

// 示例 5：稀疏数组（结果长度为5，索引1和3为空槽）
[1, , [2, , 3]].myFlat(1)
// → [1, 2, , 3]（保留空槽，实际长度5）

// 示例 6：空数组
[].myFlat(Infinity)
// → []

// 示例 7：无嵌套
[1, 2, 3].myFlat(1)
// → [1, 2, 3]

// 示例 8：深度嵌套的空数组
[[[], [[]]]].myFlat(Infinity)
// → []
```

---

## 约束

1. **不能使用** `Array.prototype.flat` 或 `Array.prototype.flatMap`
2. **支持稀疏数组**：空槽（empty slots）应保留，不能填充 `undefined`
3. **处理边界情况**：
   - `depth` 为负数时，返回原数组的浅拷贝
   - `depth` 为 `NaN` 时，按 `0` 处理
   - 非数组元素直接保留
4. **返回新数组**，不修改原数组
