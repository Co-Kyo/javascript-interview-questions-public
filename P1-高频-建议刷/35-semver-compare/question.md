# 35 - 版本号比较 (Semver Compare)

## 分类
工程实用场景

## 难度
⭐⭐

## 考察点
字符串解析、逐段数值比较、语义化版本规范理解、边界处理

---

## 题目要求

实现 `compareVersions(v1, v2)` 函数，比较两个语义化版本号字符串：

- `v1 > v2` → 返回 `1`
- `v1 === v2` → 返回 `0`
- `v1 < v2` → 返回 `-1`

### 基本约束
- 支持 `x.y.z` 格式（如 `1.2.3`）
- 支持不同长度（`1.2` 与 `1.2.0` 视为相等）
- 各段按**数值**比较（`1.10.0 > 1.9.0`）

### Bonus
- 支持预发布标签：`alpha < beta < rc < 正式版`

## 示例

```javascript
compareVersions('1.2.3', '1.2.4')   // → -1
compareVersions('2.0.0', '1.9.9')   // → 1
compareVersions('1.0.0', '1.0.0')   // → 0
compareVersions('1.2', '1.2.0')     // → 0
compareVersions('1.10.0', '1.9.0')  // → 1
compareVersions('1.0.0-alpha', '1.0.0-beta')  // → -1
compareVersions('1.0.0-alpha', '1.0.0')        // → -1
```

## 约束条件

- 输入为合法的非空版本号字符串
- `split('.')` 按 `.` 拆分版本号各段
- 注意将字符串段转为数字再比较
